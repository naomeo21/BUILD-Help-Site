Customize Profile

In the user settings, you can customize your profile by adding contact information and a photo. You can also change your password.

Prerequisites

When you first join, an SAP Cloud Identity is created for you. After you activate your account, you can sign in and customize your profile. Your initials appear in a circle on the top right. To edit your profile, select the circle, then User settings.

Set Contact Information

Procedure

  1. In the User settings, select Contact Info.
  2. Make changes to your contact information as needed.
  3. Select Save changes.

Upload Photo

Procedure

  1. In the User settings, select Photo.
  2. To upload a photo, select Upload Photo and select a photo from your computer.
  3. Drag the circle on the magnification bar to zoom and reposition your photo.
  4. When you are done editing the photo, select Crop and use.
  5. Select Save changes.

Select Organization

Procedure

  1. In the User settings, select Organization.
    Note The first time you sign in to an organization, the system automatically sets the current organization as the default organization. You can change to another organization if you wish.
  2. From the list under Default Organization, select the organization that you want to work in. The list displays the standard organization as well as all the organizations you have joined so far. If you use the organization-agnostic URL http://www.experiencesplash.com, you will be redirected to your default organization.
    Note

    If you use an organization-specific URL such as http://[organization].experienceplash.com, the default organization will not be used.

  3. Save your selection.

    The change in the default organization is applied from the next time you log in.
