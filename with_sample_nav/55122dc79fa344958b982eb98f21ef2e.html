Add UI Controls

The UI Editor includes many built-in UI controls to make prototypes interactive. Simply drag controls to the canvas and apply properties and actions to customize their function.

Prerequisites

Before you can add UI controls to a prototype, be sure you have an image on a page in the UI Editor.

Procedure

  1. Open your project and one of your prototype pages. You are now in the UI Editor.
  2. On the Controls tab, drag a control onto the canvas where you want it on your image.
  3. With the control still selected on your image, you can change the Properties and Actions in the Properties panel on the right.
    • Properties: Properties control how the UI element looks. Select the UI element and change the properties to meet your requirements.
    • Actions: Actions control what the UI element does when the user selects it. Select the UI element and change how you want the UI element to interact.
    Note For more information, see UI5 Development Toolkit.
  4. Select the Preview button on the toolbar to preview your page.

    Previewing your prototype displays the prototype pages and interactions as users will see them when using your application.

  5. Select buttons, hotspots, and other controls that you added to the pages, and test the actions.
  6. Select the device at the top to view your pages in other formats like tablet, phone, or desktop.
  7. Select the Editor button on the toolbar to return to the UI Editor.
