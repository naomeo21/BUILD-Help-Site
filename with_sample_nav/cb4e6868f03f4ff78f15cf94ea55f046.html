Control Properties in the Smart Template Object Page Template

This table summarizes how properties are displayed in the property pane:

Location

Property Name

Description

Header

Header Bar Text

The text displayed in the header bar at the top of the page. Bindable.

Header

Title

Title of the object page, usually the name/ID of the current object. Bindable.

Header

Description

The description of the Object Page. Bindable.

Header

 Image Source

See Use Images with Smart Template Object Pages

The points to the image that will be displayed in the header. The image must be referenced in the data model, assets/imagename>.jpg. Bindable.

KPI

Title

The title of the KPI, displayed at the top of the KPI.

Displayed in the outline.

KPI

Value

The value of the KPI, displayed on the left. Bindable.

KPI

Currency

The currency or the unit for the KPI, displayed on the right. Bindable.

Action Button

Label

Label of the action button.

Displayed in the outline.

Section

Title

The title of the section.

Displayed in the outline.

Form Group

Title

The title of the form group. Displayed in the outline.

Form Element

Label

The label of the form element.

Displayed in the outline.

Form Element

Value

The value of the form element. Bindable.

Form Element

Currency

The currency value of a form element.

Form Element

Mandatory

Specifies if the element is mandatory in edit mode in the form. If Yes an asterisk is displayed left of the label in edit mode.

Form Element

Read-only

False: the element is read-write

True: the element is read-only.

Form Element

Edit Mode
Lets you chose among three modes of editing data in the form element:

  • Standard input

  • Dropdown list

  • Value-help dialog box

Column

Column Header

Displayed in the outline. Not bindable.
Column

Value

Value displayed in the column. Bindable.

Column

Currency

 If this property is specified, its value is concatenated after Value and is displayed as a currency. Bindable.

 Allow navigation to another Object Page via proposing the navigation links in 1 to 1 relationships from the object bound to the table. Bindable.

Column

Boldface value

 A value displayed in bold typeface above the first value in the column.

Column

Responsiveness

 Specifies the responsive behavior of the column:

  • Fixed: the column is always displayed

  • Pop-in: the column appears in the pop-in area when the size decreases

  • Pop-out: the column is hidden when the size decreases
Not bindable
