Add Responsive Pages

The UI Editor in the Prototype tool lets you quickly create responsive pages. Add a responsive page to build cross-platform apps in a responsive page layout for use in various formats: phone, tablet, and desktop. Responsive pages automatically arrange controls on the canvas for the the appropriate device.

Prerequisites

Before you can start creating pages, you must first start a new project. See Create a New Project.

Procedure

  1. Open your project.
  2. Select Start Prototyping.
  3. Select the page style Responsive and select Create a Page.

    The page map displays with a thumnbail of the page you just added.

  4. In the page map view, select New Page, then select the page style, to add more pages.
  5. Select a page to open it in the UI Editor.
  6. To add more pages, select New Page +, in the Outline tab on the left panel and select the page style you want to add.
  7. Use the Outline in the content panel to work with pages and content:
    • Rename Page: To rename a page, select the page name and type in a new name.
    • Delete Page: To delete a page, select the page name and select Delete or Backspace. All content on that page is deleted. To undo a delete, select the undo button on the top toolbar.
    • Add Page: Select New Page + and select the page style you want to add to add.
  8. Add an image to your page in any of these ways:
    • Drag an image from your computer to the canvas.
    • Drag an uploaded image from the Files tab in the UI Editor to the canvas.
    • Upload an image using the Files tool in the Toolpicker, then drag it from the Files tab to the canvas. See Project Files for details.
  9. To resize the image, select it, and use the Scaling properties on the Properties panel on the right.
    Note

    You can now add UI Controls and then create or import a data model.
