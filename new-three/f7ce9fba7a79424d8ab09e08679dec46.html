Add a Project Picture

As project owner you can display a picture on the project tile. Adding a unique picture to each of your project thumbnails makes it easier to identify the project at a glance, and lends your project a visual identity.

Prerequisites

Sign in to SPLASH, and create a project.

Procedure

  1. On the Home page, select the camera icon on the thumbnail of a project.
    The Select from files dialog is displayed.
  2. Click Upload From Your Computer to browse for and upload an image.
  3. In Available File section, select the image.
    The file is selected.
  4. Select Select.

Results

The image is displayed on the project thumbnail. All invited members also see the picture.
