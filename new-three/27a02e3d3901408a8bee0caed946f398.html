Smart Template Object Page

The Smart Template Object Page displays all relevant information about an object.
It contains a header and two sections:
  • A form section that includes form elements an form groups
  • A table section

If you want to display sections that contain other UI elements than a form or a table, use an Object Page instead of a Smart Template Object Page.

As with the list report template, the application lets you add controls and sections by dragging them to the Outline; you add properties in the Properties pane.

Note

As you add controls and properties, the changes are displayed in the Preview pane. You cannot make changes in the Preview pane, however.

Preview pane of the object page

The Object Page template contains the following controls:
  • In the header:
    • KPI
    • Action button
  • In sections:
    • Form
    • Table in a section
  • In Multi-Forms: Groups
  • In Groups: Form Element
  • In Table: Column
