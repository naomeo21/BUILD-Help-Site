Install the BUILD Plugin in Web IDE

After you've created the destinations, wait a few minutes for the BUILD plugin to become available in Web IDE.
Note

You install the BUILD plugin one time per user.

Prerequisites

Locate your Web IDE URL. You can find it in the HCP Cockpit by selecting Subscriptions from the menu on the left. The link appears in the Active Version column of the table on that page.

Procedure

  1. Launch Web IDE, then select  Tools  Preferences .
  2. Select  Plugins   Optional Plugins .
  3. Under Plugin Repository, select BUILD Destination from the list.
  4. Enable the import_build_prototype plugin, then Save.

  5. Refresh your browser.
    Note

    Your changes will not be displayed until you have refreshed the browser.
