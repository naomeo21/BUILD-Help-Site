Add Master Detail Pages

The UI Editor in the Prototype tool lets you quickly create split-screen layout Master Detail pages. These pages include two structured pages with built-in navigation. One page displays a list of selectable items (master list). The second page displays the details of the selected item (details area).

Prerequisites

Before you can start creating pages, you must first start a new project. See Create a New Project.

Procedure

  1. Open your project.
  2. Select Start Prototyping.
  3. Select the page style Master Detail and select Create a Page.

    The page map displays with a thumnbail of the two pages you just added: Master Page 1 and Detail Page 1.

  4. In the page map view, select New Page, then select the page style, to add more pages.
  5. Select a page to open it in the UI Editor.
  6. To add more pages, select New Page +, in the Outline tab on the left panel and select the page style you want to add.
  7. Use the Outline in the content panel to work with pages and content:
    • Rename Page: To rename a page, select the page name and type in a new name.
    • Delete Page: To delete a page, select the page name and select Delete or Backspace. All content on that page is deleted. To undo a delete, select the undo button on the top toolbar.
    • Add Page: Select New Page + and select the page style you want to add to add.
  8. Select Master Page 1 in the Outline (or the left side on the canvas) to customize the List Page Properties and Actions by adding values in the right panel. Change or add Attributes under Children as needed.
  9. Select Detail Page 1 in the Outline (or the right side on the canvas) to customize the List Page Properties and Actions by adding values in the right panel. Change or add Attributes under Children as needed.
    Note

    You can now add UI Controls and then create or import a data model.
