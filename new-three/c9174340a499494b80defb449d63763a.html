Navigation with Smart Templates

Navigation in smart templates can be either implicit or explicit.

Implicit Navigation

Navigation is implicit in Smart Templates when a Smart Template List Report page and a Smart Template Object page contain the same main object.

Examples are as follows:
  • Navigation from a table in the Smart Template List Report page to the Smart Template Object Page.
  • Navigation from a table section in a Smart Template Object Page to another Smart Template Object Page.
Explicit Navigation

There is a bindable URL property for table columns and form elements. When selecting the binding, a list of navigation possibilities from the object bound to the table or page is displayed. The cardinality of the object is a 1-to-1 relationship.

Examples are as follows:
  • Navigation from a list report link column to and object page, with context.
  • Navigation from a link column in a table in an object page to another object page, with context.
  • Navigation from a form value with a link in an object page to another object page, with context.
