What is SPLASH?

Splash is a User Experience (UX) service that provides design and user experience best practices, tools, and e-learning to organizations looking to implement a design-led and user-centered approach to innovation.

Use SPLASH to create prototypes, do user research, learn design concepts, and to scale and accelerate the development process.

SPLASH is designed for:

  • LEARNING: Learn design concepts and best practices method cards.

  • DOING: Prototyping tool includes the UI Editor and Data Editor to create interactive prototypes without writing a single line of code. The Research Tool lets you easily gather user feedback on prototypes to ensure you build the applications users want.

  • INSPIRING: Gallery includes sample apps you can use to create your own customized apps. You can also publish your own apps to the Gallery and share with others.
