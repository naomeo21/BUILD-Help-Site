The Edit Feature

By default, the Smart Template Object Page opens in read-only mode, however, the Edit feature permits changes in an object page without making changes in the data model.

You determine the way data is edited when you configure the first section of an object page. Input can be as follows:
  • Standard Input
  • Dropdown list
  • Value Help Dialog box (F4)
The Edit link is in the upper-right corner of the Smart Template Object Page.
You can disable the Edit feature by editing the data model and setting the objects as read-only as follows:
  1. In the Data Editor, select the object under the Properties tab.
  2. Under the Info tab, select Read-Only.
