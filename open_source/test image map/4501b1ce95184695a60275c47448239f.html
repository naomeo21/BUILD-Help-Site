Bind Data to UI Elements

When you bind data, you are making a link between the user interface elements in your prototype pages to the business logic in the data model.

Prerequisites

Before binding data to your user interface, you must first create or import a data model into your project. If you aren't using a data model, you do not bind data. You can manually enter the data on your pages.
Note

These steps apply to binding data only on Freestyle, Responsive, Master Detail pages. Smart Template pages are less configurable than standard pages. See Binding for information on binding Smart Template pages.

Procedure

  1. Open your project, then open one of your prototype pages.
  2. In the UI Editor, select the Data tab. Make sure you have a data model associated with your pages.
  3. Note If you are using a list or collection on your first page, select a main object on pages after the first page. (different from Smart Templates).
    dsfgdfgsdfgsdfgdfg
  4. You can bind data from the Data Editor tab in the UI Editor in one of these ways:
    • Drag an object or property from the Data tab to the UI control on the canvas.
    • Drag an object or property from the Data tab to the property in the Properties pane on the right.
    • Select the object property from the dropdown list in the Properties pane.
