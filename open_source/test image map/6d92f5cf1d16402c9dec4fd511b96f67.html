Create Feedback Studies

You can create multiple feedback studies in each project. First you add a name and description of the study. You add images to the study, and then add questions about those images for participants to answer. You can add HTML prototypes to the study, and create tasks for participants to complete using the prototypes. Then you publish the study and invite people to take part.

Prerequisites

To create a feedback study, you must have previously created a project, or be a member of an existing project.

Procedure

  1. On the Home page, select a project tile.
    Figure 1: A sample Home page.

    The Project page is displayed.

  2. In the Study section, select Create New Study.

    The study name and description dialog is displayed.

  3. Enter a name and description for the study.

    For more information, see Guidelines for Study Names and Descriptions.

  4. Select Save.

    The Feedback tool page is displayed. To add content for study participants to engage with, you add questions and tasks to your study.

  5. Create a task or ask a question.
  6. You can select and drag image and task thumbnails on the study page to re-order them.
  7. Select the Preview icon to preview your study.

    The preview is displayed on a new tab, showing you how the feedback study will appear to its participants..

  8. To publish the study, select Publish.

Results

A unique study-link for your feedback study is displayed, and you can send the study-link to people that you want to provide feedback.
