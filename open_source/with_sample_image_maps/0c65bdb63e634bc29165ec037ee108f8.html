Editing Data in the Data Modeler

You can add data to a newly-created object or edit data in an existing object in the Data Editor from the Samples tab.

Procedure

  1. In the Data Editor, select the object you want to edit, then select Samples.
  2. Select Open Editor.

    The editor displays the name of the object on the tab and the properties as column headers.

  3. Make any required changes in the table cells or header.
  4. If you want to add data, select the + plus sign in the upper left of the table to add new rows.
  5. When you are finished with your additions or edits, select OK.
