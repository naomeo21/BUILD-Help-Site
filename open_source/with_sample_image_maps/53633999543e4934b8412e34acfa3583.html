Mixing Smart Template and Non-Smart Template Pages

You create prototypes mixing Smart Template pages with other types of pages.

You can create prototypes using only Smart Template pages -- Smart Template List Report and Smart Template Object Pages -- but you can also create prototypes mixing Smart Template Pages with non-smart template pages under the following conditions:
  • The first page must be a Smart Template List Report page, which must have a main object.
  • Only one other page in a smart template prototype may have the same main object as the main object of the List Report Page.
  • Aside from the List Report page, you may not have two object pages with the same main object.
  • You can use the following types of pages in your smart template prototype:
    • Freestyle
    • Responsive
    • Object page
