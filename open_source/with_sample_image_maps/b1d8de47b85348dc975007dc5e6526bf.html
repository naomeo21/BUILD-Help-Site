Add Master Detail Pages

The UI Editor in the Prototype tool lets you quickly create a split-screen layout using Master Detail pages. These pages include two structured pages with built-in navigation. One page displays a list of selectable items (master list). The second page displays the details of the selected item (details area).

Prerequisites

Before you can start creating pages, you must first start a new project. See Create a New Project.

Procedure

  1. Open your project.
  2. Select Start Prototyping.
  3. Select the page style Master Detail and select Create a Page.

    The page map displays with a thumbnail of the two pages you just added: Master Page 1 and Detail Page 1.

  4. Select a page and open it in the UI Editor.

    In the UI Editor, you see the Master page on the left and the Detail page on the right. Select either side of the page to customize your pages.

    Figure 1: Master Detail Pages in UI Editor
  5. Use the Outline in the left content panel to work with pages and content.

    Action

    Steps
    Add Page Select + to add any of these pages:
    • List Page (Master)
    • Responsive (Detail)
    • Object Page (Detail)
    • Icon Tab Bar Page (Detail)
    Rename Page Select the page name and type in a new name.
    Delete Page Select the page name and then or Backspace. All content on that page is deleted. To undo a delete, select the undo button on the top toolbar.
    Undo/Redo Changes

    Select or your recent changes.
  6. Select Master Page 1 in the Outline (or the left side on the canvas) to customize the List Page Properties and Actions by adding values in the right panel. Change or add Attributes under Children as needed.
  7. Select Detail Page 1 in the Outline (or the right side on the canvas) to customize the List Page Properties and Actions by adding values in the right panel. Change or add Attributes under Children as needed.
    Note

    You can now Add UI Controls and make your prototype interactive.
