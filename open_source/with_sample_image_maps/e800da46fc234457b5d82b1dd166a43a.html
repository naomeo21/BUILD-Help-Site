Create a Project by Cloning

You can create new projects by cloning projects created and shared by other SPLASH users. You can do this from your SPLASH HOME page.

Prerequisites

Sign in to SPLASH.

Procedure

  1. On the Home page, select a project tile.

    Select to expand image.

    Figure 1: AN image

    The Project page is displayed.

  2. On your project Home page select one of the sample projects.

    The Gallery Project page for the project is displayed.

    Figure 2: Click to view the Gallery Project page for a a sample project.
  3. Select Clone.

    The project is cloned.

Results

The cloned project and clones of the public files in the project are displayed on your Project page. You can invite team members, create feedback studies, and modify the prototype pages. The following details are not cloned:
  • The date and time that project files were uploaded.
  • The names of the SPLASH users who uploaded files.
  • The names and details of the original SPLASH users who owned the project or were team members on the project.
