Set Things Up

This section describes how to set up and activate the BUILD plugin in your HANA Cloud Platform (HCP) account.

Before you start, log in to Splash and locate the name of the Splash organization you are using. It is the name that is displayed in the Organization screen of the user settings.

You need to perform two manual actions on your HCP account to access BUILD prototypes through Web IDE.
The first is to create the destinations for the plugin and for every organization to which you want to be able to import prototypes.
Note

You need to create destinations one time per company.

Create the destination for the Web IDE plugin
In the SAP HANA Cloud Platform Cockpit, select  Destinations  New Destination .
  1. Enter or select the following information:
    • Name: Web_IDE_PLUGIN
    • Type: HTTP
    • Description: BUILD Destination
    • URL: https://www.experiencesplash.com/webideplugin
    • Proxy Type: Internet
    • Authentication: NoAuthentication
  2. Under Additional Properties add the following properties and their values:
    • WebIDEEnabled true
    • WebIDEUsage plugin_repository
  3. Save the parameters.

Create the destination for your organizations
  1. Create a second destination in the HCP Cockpit using these parameters::
    • Name: <your Splash organization>_TENANT for example, standard
    • Type: HTTP
    • Description: Destination for <Your Splash organization>_TENANT for example, Standard_TENANT
    • URL: https://www.experiencesplash.com/webide/<Your Splash organization>, for example, https://www.experiencesplash.com/webide/standard
    • Proxy Type: Internet
    • Authentication: NoAuthentication. You will be prompted for your BUILD user ID during import.
  2. Under Additional Properties add the following properties and their values:
    • WebIDEEnabled true
    • WebIDEUsage build_system
  3. Save the parameters.

Create a Destination for SAP UI5
If you are an SAP Employee, you must create a third destination in the HCP Cockpit with the following information:
  • Name: sapui5-private
  • Type: HTTP
  • Description: Private version of UI5 for WebIDE
  • URL: https://www.experiencesplash.com/staticcontent/sapui5private
  • Proxy Type: Internet
  • Authentication: NoAuthentication
