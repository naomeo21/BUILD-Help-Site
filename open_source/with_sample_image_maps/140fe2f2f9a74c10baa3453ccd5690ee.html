Use Images with Smart Template Object Pages

You can use images in the header of Smart Template Object Pages to make them more pertinent or attractive.

This section applies to Smart Template Object Pages, only, and does not refer to using images in other types of prototype pages.

Rules for Using Images with Smart Template Object Pages

Because you are going to bind the image to a property, you must respect the following rules:
  • You can only use a data model that has been generated from an Excel sheet.
  • The images must be referenced in the data model as follows: assets/image for example, assets/Bob.jpg

  • The images must be uploaded to the application using the Files Tool.

  • When searching for the image, you must use the same syntax as in the Excel sheet: assets/image
  • You must bind the image to the object image property.

Prerequisites

You are creating a Smart Template Object Page

Procedure

  1. In the main Splash header, select PrototypeFiles.

  2. Do one of the following:
    • Drag the files you want to upload from your computer to the Files page.
    • Select the Upload Files link.

  3. Return to you Smart Template Object Page, then configure the Header.
  4. In the Image Source box, select the Image property.
  5. Select the Files tab.

    The image that corresponds to the property that you selected appears in the header.
