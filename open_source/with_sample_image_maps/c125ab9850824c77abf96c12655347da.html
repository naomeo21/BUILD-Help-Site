Add Properties to an Object

You can add properties to an object from the Properties tab on the right side of the editor.

Prerequisites

You have uploaded or manually created one or more objects.

Procedure

  1. In the Data Editor, select Properties.
  2. Select New Property, then type the name of the property in the field.

    Make sure that you follow the naming rules:

    • The maximum length is 40 characters.
    • The first character must be a letter or an underscore.
    • Subsequent characters must be alphanumeric or an underscore. No spaces are allowed.
    • The name must be different from names of other properties of the object.
  3. Select the type of the property from the list. The standard values are:
    • The standard values are:
      • String - the default value
      • Numeric
      • Date - Dates are displayed in the system format, which is YYYY-MM-DD.

    • Additional property types are:
      • Boolean
      • Binary
      • Byte
      • Double
      • Single
      • Guid
      • Int16
      • Int32
      • Int64
      • SByte
      • Time - Time is displayed as HH:MM:SS AM/PM, and the time zone in UTC.
      • DateTimeOffset
