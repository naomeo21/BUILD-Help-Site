Extend Prototypes with SAP Web IDE

SAP Web IDE is a browser-based development tool that helps you create and extend end user applications for browser and mobile devices.

Web IDE simplifies the end-to-end application life cycle: development, packaging, deployment, and customer extensions for SAPUI5 applications.

BUILD generates the code that you can import into Web IDE as a starting point for developers. The same code that is generated for your BUILD prototype, used in previewed and published applications, can be imported into SAP Web IDE using the BUILD plug-in. This can then be used as a starting point for developers, to add additional logic to the design and to connect to the existing oData services. You can use Web IDE to connect to the existing backend services, and you can see data from the server.
Prerequisites
To use Web IDE with BUILD prototypes, you need the following:

  • An SAP HANA Cloud Platform (HCP) developer account to work with Web IDE. Register for an HCP account. Log on to the SAP HANA Cloud Platform Cockpit.

  • A Web IDE instance. This may be a trial version or a version licensed by SAP.
  • A BUILD prototype published in BUILD.

For information on signing up for a developer account, see Signing Up for a Developer Account

For information on using Web IDE, see SAP Web IDE in the SAP HANA Cloud Documentation

High-level Steps
Below is a high-level overview of what you need to do to get started. Each step is explained in detail in the next sections:
  1. Configure the SAP HANA Cloud Platform (HCP) to create destinations for Web IDE-to-BUILD systems.
  2. Enable the BUILD plugin in Web IDE.
  3. Import BUILD prototypes.
