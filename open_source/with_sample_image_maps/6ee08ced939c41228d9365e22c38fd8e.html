Add Prototype Pages

The UI Editor in the Prototype tool lets you quickly create various types of prototype pages to meet your specific needs.

You can create several types of prototype pages.
  • Freestyle:: Start with a blank page, which is perfect for prototyping without positioning constraints.

  • Responsive: Similar to the freestyle page, but ideal for prototypes that need to look great on different devices.

  • Object: A predefined smart template page. Create a page designed to view, add, or edit any object using either a tab or flat view.
  • List Report: Allows the user to work with a large list of items and take action. This floorplan combines powerful functions for filtering large lists with different ways of displaying the resulting item list. You can only use the list report floorplan with the full screen layout.
  • Master Detail: Create two structured pages with built-in navigation. One page displays a list of selectable items. The second page displays the details of the selected item.
