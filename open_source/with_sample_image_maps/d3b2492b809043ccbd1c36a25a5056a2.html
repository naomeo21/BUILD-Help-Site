Create a Project in Web IDE

After you've installed the BUILD plugin, you are ready to create a project in Web IDE. Select the Home icon. From the home page do the following:

Procedure

  1. Under Create a Project, select New Project from Template.
  2. On the next page, select BUILD Project, then select Next.
  3. Give your project a name, then select Next.
  4. If you are prompted, enter your Splash user name and password.
  5. Under Available Prototypes, select the BUILD prototype that you want to run in Web IDE, then select Next.
  6. Select Finish.

    A new project is created in your workspace.

    You can see your prototype imported as UI5 code, and you can immediately preview the app to see the BUILD prototype in action from the code. From here, you can modify or enhance the design via code, connect to real data sources such as oData, and then test the changes by running the application in Web IDE.

    To run the prototype in Web IDE, select Run.
