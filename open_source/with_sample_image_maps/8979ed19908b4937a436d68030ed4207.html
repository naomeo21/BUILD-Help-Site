Edit Data

You can disable the Edit feature by editing the data model and setting objects as read-only.

You determine the way data is edited when you configure the first section of an object page. Input can be as follows:
  • Standard Input
  • Dropdown list
  • Value Help Dialog box (F4)
The Edit link is in the upper-right corner of the Smart Template Object Page.

Procedure

  1. From the object page that you want to edit, select Preview.
  2. Select Edit in the header.
  3. Make the required changes, then select Save. The modified data appears in the object page.
