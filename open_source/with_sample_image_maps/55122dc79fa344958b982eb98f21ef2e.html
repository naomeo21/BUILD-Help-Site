Add UI Controls

The UI Editor includes many built-in UI controls to make prototypes interactive. Simply drag controls to the canvas and apply properties and actions to customize their function.

Prerequisites

Before you can add UI controls to a prototype, be sure you have an image on a page in the UI Editor.

Procedure

  1. Open your project and one of your prototype pages.
  2. In the UI Editor, on the Controls tab, drag a control onto the canvas where you want it on your image. You can also enter a specific control in the search box to search for it.
  3. Select the control on your image and customize what the controls do in your prototype in the Properties panel on the right.
    • Properties: Properties control how the UI element looks. Select the UI element and change the properties to meet your requirements.
    • Actions: Actions control what the UI element does when the user selects it. Select the UI element and change how you want the UI element to interact.
    Note For more information, see UI5 Development Toolkit.
