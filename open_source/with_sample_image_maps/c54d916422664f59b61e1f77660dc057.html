Get Started with SPLASH

SAP Splash is a user experience (UX) service that provides design and user experience best practices, tools, and e-learning to organizations looking to implement a design-led and user-centered approach to innovation.

There are a few tasks you need to complete before you can get up and running using Splash. If you want access to all the advanced Splash features, you must first create a Splash account. It's free to just browse and do a few tasks. The rest requires an account.
